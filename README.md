# Pulse — App Launch Landing Page

A vibrant fitness app launch page demo built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

> Deploy to Vercel and paste your URL here:

## Features

- Responsive design with phone mockup UI
- App Store / Google Play download buttons
- Features, stats banner, testimonials
- Free trial CTA section
- Interactive buttons with toast feedback

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

```bash
npm run build
npm run preview
```
